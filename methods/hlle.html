Hessian Locally Linear Embedding

Just like the Local Tangent Space Alignment, the Hessian Locally Linear Embedding algorithm is very similar to the Locally Linear Embedding algorithm. Given a set of feature vectors \(X = \{ x_1, x_2, \dots x_N \} \) the HLLE algorithm proposes to perform the following steps:
  1. Identify nearest neighbors.
    For each \( x \in X \) identify its \( k \) nearest neighbors, i.e. a set \( \mathcal{N}_x \) of \( k \) feature vectors such that \begin{equation} \arg\min_{\mathcal{N}_x} \sum_{x_n \in \mathcal{N}_x}\| x - x_n \|_2^2 \label{nearest_neighbors}\end{equation}
  2. Analyze hessian of a patch.
    For each \( x \in X \) compute the Gram matrix \( G \) of its neighbors such that \( G_{i,j} = (\mathcal{N}_x^{i} , \mathcal{N}_x^{j}) \) and center it (just like in KLLE). Compute its \( t \) (the number of required features) eigenvectors \( v_1, \dots v_t \). Construct hessian approximating matrix \[ Y = \begin{bmatrix} 1_k & v_1 & \dots & v_t & v_1\cdot v_1 & \dots & v_1\cdot v_t & \dots \end{bmatrix}, \] where \( \cdot : X \times X \to X \) denotes coefficient-wise product. Normalize columns of the matrix \( Y \) and then compute matrix \[ Q = Y Y^{T}\] and put it to the sparse alignment matrix \( L \) (initially set by zeroes) using the following equation: \[ L \leftarrow L + Q. \]
  3. Embedding through eigendecomposition.
    To obtain \( t \) features (coordinates) of embedded vectors solve the partial eigenproblem \[ L f = \lambda f, \] for smallest eigenvalues \( \lambda_1, \dots, \lambda_t, \lambda_{t+1} \) and its corresponding eigenvectors \( f_1, \dots, f_t, f_{t+1} \). Drop the smallest eigenvalue \( \lambda_1 \sim 0 \) (with the corresponding eigenvector) and form embedding matrix such that \(i\)-th coordinate (\( i=1,\dots,N \)) of \(j\)-th eigenvector (\( j=1,\dots,t\) ) corresponds to \(j\)-th coordinate of projected \(i\)-th vector.

Tapkee implementation

To use the HLLE algorithm put HESSIAN_KERNEL_LOCALLY_LINEAR_EMBEDDING value as REDUCTION_METHOD to the parameter map. You can also set number of neighbors \( k \) as NUMBER_OF_NEIGHBORS in the parameter map and the eigenproblem shift can be set using EIGENSHIFT key. To change the method of obtaining nearest neighbors use NEIGHBORS_METHOD (recommended value is COVER_TREE) key of the parameter map and to change the method of eigendecomposition use EIGEN_EMBEDDING_METHOD (recommended value is ARPACK).
The high-level implementation of the Hessian Locally Linear Embedding algorithm can be found here.
