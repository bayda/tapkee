Stochastic Proximity Embedding

Stochastic Proximity Embedding (SPE) is an efficient algorithm for generating low-dimensional Euclidean embeddings given a set of feature vectors \(X = \{ x_1, x_2, \dots x_N \} \) and/or their symmetric distance matrix \( D_{ij} \). It is an interesting method because of its simplicity and efficiency, it scales linearly with the sample size \( N \).

SPE consists of the following steps:

  1. Choose an initial learning rate \( \lambda \). This parameter will decrease later in order to avoid oscillatory behaviour.

  2. Initialize randomly the point coordinates in the embedded space \(Y = \{ y_1, y_2, \dots y_N \} \).

  3. Update a set of the point coordinates.
    For a prescribed number of iterations \( S \), choose a pair of point indices \( i \) and \( j \) and compute their distance in the embedded space, \[ d_{ij} = || x_i - x_j || \] If \( d_{ij} \neq D_{ij} \) then update the coordinates of the selected points by
    \[ x_i \leftarrow x_i + \lambda \frac{1}{2} \frac{D_{ij} - d_{ij}}{d_{ij} + \epsilon} (x_i - x_j) \] \[ x_j \leftarrow x_j + \lambda \frac{1}{2} \frac{D_{ij} - d_{ij}}{d_{ij} + \epsilon} (x_j - x_i) \]

  4. Decrease the learning rate \( \lambda \) by a quantity \( \delta \lambda \).

  5. Repeat steps 2 to 4 for a predetermined number of iterations \( C \).

Tapkee implementation

Stochastic Proximity Embedding is used by means of setting the value of REDUCTION_METHOD to STOCHASTIC_PROXIMITY_EMBEDDING in the parameter map. The high-level implementation of the Stochastic Proximity Embedding algorithm can be found here.
